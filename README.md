# Coffee-Table

# Overview

This project explores transactional data from Bright Coffee Shop to deliver meaningful insights for the newly appointed CEO. It focuses on identifying key revenue sources, peak sales periods, and product performance patterns to guide strategic business decisions.

# Objectives

1.Determine which products generate the most revenue

2.Identify peak sales times throughout the day

3.Analyze sales trends across product categories and time intervals

4.Recommend strategies to improve sales performance and operational efficiency

# Tools used

  1. SQL- To clean, transform and analyze data
  2.  Snowflake- for queries
  3. Microsoft Ecxel- To review my data
  4. Google looker- For data visuals and create dashboards
